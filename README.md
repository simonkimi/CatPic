# Catpic
使用flutter编写的图片采集软件

可以从多种图站采集图片

支持中国大陆地区直连

# 功能
- [x] 多种类型网站支持
- [x] Post显示功能
- [x] 中国大陆地区直连
- [x] 图片预览
- [x] Tag显示功能
- [x] 历史搜索预览
- [x] Tag自动补全
- [x] 选择图片质量
